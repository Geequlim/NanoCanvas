Enumerator
GenerateMipmaps 

Generate mipmaps during creation of the image.

RepeatX 

Repeat image in X direction.

RepeatY 

Repeat image in Y direction.

FlipY 

Flips (inverses) image in Y direction when rendered.

PreMultiplied 

Image data has premultiplied alpha.
